# Final Design Colors - Masterly App

## Color Palette (Modern Educational App Theme)

### Primary Gradients
```
Main Background: #6366F1 (Indigo) → #8B5CF6 (Purple) → #A855F7 (Violet)
Alternative: #3B82F6 (Blue) → #6366F1 (Indigo)
```

### Component Colors
```
Glass Card: rgba(255, 255, 255, 0.15)
Glass Border: rgba(255, 255, 255, 0.25)
Backdrop Blur: 16px
```

### Button Gradients
```
Primary (Success): #10B981 (Emerald) → #059669 (Green)
Secondary (Info): #3B82F6 (Blue) → #2563EB (Blue)
Danger (Error): #EF4444 (Red) → #DC2626 (Red)
Warning: #F59E0B (Amber) → #D97706 (Orange)
Purple: #A855F7 (Purple) → #EC4899 (Pink)
```

### Text Colors
```
Primary: #FFFFFF (White)
Secondary: rgba(255, 255, 255, 0.85)
Muted: rgba(255, 255, 255, 0.65)
Disabled: rgba(255, 255, 255, 0.45)
```

### Accent Colors
```
Success: #10B981
Warning: #F59E0B
Error: #EF4444
Info: #3B82F6
Purple: #A855F7
Pink: #EC4899
Cyan: #06B6D4
```

### Badge/Card Gradients

1. #FBBF24 (Yellow) → #F59E0B (Amber)

## Asset Placeholders

### Icons (to be placed in `/public/assets/icons/`)
- logo.svg - App logo
- logo-white.svg - White version for dark backgrounds
- icon-192.png - PWA icon 192x192
- icon-512.png - PWA icon 512x512
- favicon.ico - Browser favicon

### Mobile Assets (to be placed in `/app/assets/`)
- icon.png - App icon (1024x1024)
- splash-icon.png - Splash screen icon
- adaptive-icon.png - Android adaptive icon
- favicon.png - Web favicon

### Illustrations (optional, to be placed in `/public/assets/illustrations/`)
- empty-state.svg - Empty state illustration
- success.svg - Success celebration
- error.svg - Error state
- loading.svg - Loading animation
